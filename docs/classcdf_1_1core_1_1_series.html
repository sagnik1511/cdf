Detailed Description

A class that represents a series of heterogeneous data values and provides comparison utilities.

The Series class stores a sequence of values of type _cdfVal (which can hold multiple types, such as int, double, or std::string) and allows comparisons between the series elements and a provided value using standard comparison operators (e.g., ==, <, <=, >, >=, !=).
