using _cdfVal = std::variant<std::string, int, double, cdf::NaN>

A variant type to represent different data types.

_cdfVal can hold one of the following types:

  • std::string
  • int
  • double
  • cdf::NaN

This type is useful for scenarios where the data type is not known in advance and may vary dynamically. The inclusion of cdf::NaN allows handling of missing or undefined values in computations.
