Reads a CSV file and loads data into a cdf::DataFrame.

This function opens a CSV file, processes its content, and converts it into a DataFrame object. It handles headers, row parsing, and converts data types as necessary based on the file contents.

Parameters
csvFilePathThe path to the CSV file to be loaded.
delimiterThe delimiter used to separate columns (default is comma ,).
headerThe line number that contains the column headers. If header is -1, no headers are read from the file.
namesA vector of column names to use if header is specified as -1, otherwise it uses the first line as headers.
Returns
A DataFrame object containing the data read from the CSV file.
Exceptions
std::ios_base::failureif the file cannot be opened.
